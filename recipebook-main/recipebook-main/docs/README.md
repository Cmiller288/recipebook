# My Cookbook (Front-end App)

A simple, mobile-first front-end app for browsing recipes. This project includes: login/logout demo, search, filter, sort, session storage, and a data inspection page.

See `docs/README.md` for full documentation, deployment instructions (GitHub Pages), and authorship/attribution information.

Quick start:
1. Clone this repository
2. Serve with `Live Server` or `python -m http.server`
3. Open `index.html`

Deployment: enable GitHub Pages on this repo (branch = main, folder = root). Add the deployment URL to the `docs/README.md` after enabling.
